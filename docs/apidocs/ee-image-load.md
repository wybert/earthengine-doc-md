 
#  ee.Image.load

Returns the image given its ID. 
Usage| Returns  
---|---  
`ee.Image.load(id,  _version_)`| Image  
Argument| Type| Details  
---|---|---  
`id`| String| The asset ID of the image.  
`version`| Long, default: -1| The version of the asset. -1 signifies the latest version.
