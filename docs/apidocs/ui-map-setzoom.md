Sets the zoom level of the map. 
Returns this ui.Map.
Usage| Returns  
---|---  
`Map.setZoom(zoom)`| ui.Map  
Argument| Type| Details  
---|---|---  
this: `ui.map`| ui.Map| The ui.Map instance.  
`zoom`| Number| The zoom level, from 0 to 24, to set for the map.
